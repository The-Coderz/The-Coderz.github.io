预调用脚本

      遇到随着时间变化的变量，譬如：认证key，在之前简单的变量设置就显得有些不足了。最好就是在调用脚本的时候，能自动按照规则计算出变量的值，并且把变量这只到环境变量中。Postman的Pre-request Script就是很好的解决途径。



      此时，在用{ { token } }这样的环境变量形式(和上面提到的一样)，就可以了。
