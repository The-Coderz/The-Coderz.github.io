本地提交失败、更新冲突

路径 类型 远端改动 本地改动 冲突类型 svn提示 处理方法 可选操作
test/cr1.txt 文件 DA D 树冲突 !     C test/cr1.txt
      >     local delete, incoming delete upon update		 需要检测远端是否真的删除，如果是DA，需要比对改动 C+S+D+M+MR
src/main/java/com/taobao/aone/mypackage 目录 DA DA 树冲突 A     C src/main/java/com/taobao/aone/mypackage
      >     local delete, incoming delete upon update		 执行自己的对比脚本，比对该目录下双方的不同文件即可 M+S+MR+D+C
addDir 目录 D DA 树冲突 A     C addDir
      >     local delete, incoming delete upon update		 执行自己的对比脚本，比对该目录下双方的不同文件即可 M+S+MR+D+C
shuzhai-svn-test-branch8.release 文件 M D 树冲突 D     C shuzhai-svn-test-branch8.release
      >     local delete, incoming edit upon update		 直接展示 C+D+M+S+MR
addinconflict.txt 文件 D M 树冲突 A  +  C addinconflict.tx
      >      local edit, incoming delete upon update		 直接提示 C+D+S+MR+M
pom.xml 文件 M M 文件冲突 C        pom.xml 直接展示 M+S+MR

注：M=修改,S=保存,MR=标记为已解决,D=从结果中删除,C=从源分支上拷贝该路径到结果中
