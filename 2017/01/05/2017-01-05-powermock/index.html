PowerMock

PowerMock github project

      PowerMock是一个强大的测试框架，它集成了Mockito、EasyMock等测试框架，在基于这些框架的基础上，利用修改字节码的手段mock静态方法、私有方法、初始化方法，是一种对过去熟悉的测试框架的增强扩张。

使用

  1. 引入PowerMock的jar包，需要注意不同版本的jar包依赖的JUnit版本；
  2. 建立测试类；
  3. 利用PowerMock写单测；
